Install Quickstart

Configure Database Connection

You need to have previously setup database from the cPanel.

Here is a good tutorial how to setup MySQL database in cPanel if you are not familiar with this.

  1. Login to cPanel by accessing www.yourdomain.com/cpanel and navigate go MySQL Databases.
  2. Create database first.
  3. Create user for database and setup your password.
  4. Add the user to database by selecting the database and the username.
  5. Open the file iprofile_db.sql in database folder of package files and Import the Database. It will create all required database tables and sample content.
  6. If you are workin on localhost go to the localhost/phpmyadmin then click on new database and then click on import database option then select the databse file in you mainfolder/database/iprofile_db.sql file and upload it.

Make sure you have checked All privileged when adding the user to database.

Setup Base URL

Note: You Dont need to setup base url it will generates automatically. If you want to chagne in any case follow the lines below...

Go to root directory and then go application/config/config.php and setup $config['base_url'] where iprofile Script will be installed.

You need to add your base url to this line:

              $config['base_url'] = '';

Ex. If you are installing on subdomain you will set http://www.iprofile.domain.com/

Ex. If you are installing on subfolder you will set http://www.domain.com/iprofile/

Ex. If you are installing on the main domain you need to set just http://www.domain.com/

              $config['base_url'] = 'http://www.domain.com/iprofile/';
NOTE:The url must ends with slash (/)

Site Configuration


Change Database Settings

Go to your project root directory then open: application/config/database.php then Change your database name, username and password like in the image below,

 

General Setting

Change the Site Name, Favicon, Logo, footer content in "yourdomain/admin/general_settings" file.



Change Email Setting

Change the SMTP Username and Password in "yourdomain/admin/general_settings" file.
Note: You need to use a valid domain email address. And Email will work only on live server not on localhost.




Google Recaptcha Setting

Create a Google recaptcha Api kets and add it in "yourdomain/admin/general_settings" Google Recaptcha Tab.
Note: You need to use a valid recaptcha credentials. And recaptcha will work only on live server not on localhost.




Multilanguage Configuration

If you want to add new language, go to application/language/ folder then copy any language folder file and rename it with your language name then change all the variable to your language. After that go to the admin panel and click on the languge settings and add name and directory name.

  • You can add new labels in files available inside following language directories of the source code
  • application/language/english
  • application/language/french
  • Please note that above example is for two languages only. If you have more language directories in your project, you have to add new label ($lang[‘label_name’] = “label translation”; ) with its translation in each unique language directory in relevant language file.
